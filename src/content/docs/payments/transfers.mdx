The AlphaPay Transfers API allows you to initiate instant money transfers to any bank account or mobile money wallet across 15+ African countries.

## Initiate a Transfer

**POST** `/v1/transfers`

```bash
curl -X POST https://api.alphapay.africa/v1/transfers \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 500,
    "currency": "GHS",
    "recipient_phone": "+233241234567",
    "network": "mtn",
    "description": "Invoice payment #1042"
  }'
```

### Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `amount` | number | ✅ | Transfer amount in the source currency |
| `currency` | string | ✅ | ISO 4217 currency code (e.g., `GHS`, `NGN`, `KES`) |
| `recipient_phone` | string | ✅ | Recipient's phone number in E.164 format |
| `network` | string | ✅ | Mobile network (`mtn`, `telecel`, `airteltigo`, `mpesa`, `orange`) |
| `description` | string | ❌ | Optional description shown in transaction history |
| `reference` | string | ❌ | Your unique idempotency reference (prevents duplicate transfers) |

### Response

```json
{
  "status": "success",
  "transfer_id": "txn_GH_20240715_a8f3c",
  "amount": 500,
  "currency": "GHS",
  "recipient_phone": "+233241234567",
  "network": "mtn",
  "created_at": "2024-07-15T10:24:00Z",
  "settlement_status": "completed"
}
```

## Check Transfer Status

**GET** `/v1/transfers/{transfer_id}`

<Aside type="tip">
  Listen for `transfer.success` and `transfer.failed` webhook events instead of polling for better performance.
</Aside>

## Error Codes

| Code | Description |
|---|---|
| `INSUFFICIENT_FUNDS` | Sender wallet balance is too low |
| `INVALID_RECIPIENT` | Phone number not registered on the specified network |
| `TRANSFER_LIMIT_EXCEEDED` | Amount exceeds your account's per-transaction limit |
| `NETWORK_UNAVAILABLE` | Mobile network temporarily unavailable — retry in 30s |
