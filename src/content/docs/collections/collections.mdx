AlphaPay's Collections API lets you accept payments from customers via mobile money, bank transfer, and card. Choose between a hosted checkout page (no code) or a direct API integration.

## Integration Options

<Tabs>
  <TabItem label="Hosted Checkout">
    The fastest way to start accepting payments. Generate a payment link via API and redirect your customer to AlphaPay's hosted checkout page. No frontend code required.

    ```bash
    curl -X POST https://api.alphapay.africa/v1/payment-links \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{
        "amount": 150,
        "currency": "GHS",
        "description": "Order #5042",
        "redirect_url": "https://yourapp.com/payment-success"
      }'
    ```

    The API returns a `payment_url` — redirect your customer there. AlphaPay handles the entire checkout flow.
  </TabItem>
  <TabItem label="Inline API">
    For full control over the UI, use the inline Collections API to initiate a mobile money payment prompt directly on the customer's phone.

    ```bash
    curl -X POST https://api.alphapay.africa/v1/collections \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{
        "amount": 150,
        "currency": "GHS",
        "customer_phone": "+233241234567",
        "network": "mtn",
        "description": "Order #5042"
      }'
    ```

    The customer receives a mobile money prompt on their phone. Listen for the `payment.received` webhook on completion.
  </TabItem>
</Tabs>

## Payment Statuses

| Status | Description |
|---|---|
| `pending` | Payment initiated — awaiting customer approval |
| `success` | Payment approved and funds credited to your wallet |
| `failed` | Customer declined or timed out — safe to retry |
| `refunded` | Payment was successfully refunded |

## Issuing Refunds

<Aside type="tip">
  Refunds are processed instantly. Funds return to the original payment method.
</Aside>

```bash
curl -X POST https://api.alphapay.africa/v1/payments/{payment_id}/refund \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"reason": "Customer requested refund"}'
```
